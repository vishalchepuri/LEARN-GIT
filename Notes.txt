->Install Git bash

git --version -->To check the version of git
git config --> To check the values
git config --global --list -->To get the user details.

git init --> It initializes the git repository and the branch will be master.



git init -b main -->It creates repository with main branch.

git add <file-name> -->To add to local git repository

git rm --cached <file-name> -->To remove a file from staging area.

git commit -m "<message>" -->To commit the file


So when we made the changes first we need to add that file to staging area that is <git add >  command then we need to commit it.


git log --> To check commits with details


git commit -a -m "without staging" -->It we use -a it goes to repository without staging

git diff -->It is used to track the changes in files.

git diff --staged -->To get the change in files in the staging area.

git add . -->To add all the files to staging










git branch -M main --> It creates a new branch name with main

git remote add origin <git-link> -->It is used to add the remote link

git checkout <branch-name> -->To which branch we need to push the files we need to keep that branch name.
git push -u origin main --> It is used to push the files from local repo to github in main branch(-u means upstream)



git checkout -b <branch-name> or  git switch -c <branch-name>--> To create a branch
git branch --> To list all branches and the branch which has a * it is an active and it shows only local branches
git branch --all  --> It shows local and remote branches
git branch - -->It is used to switch to previous branch

git branch -d <branch-name> -->It is used to delete branch




git merge <branch-name> -->To merge mutliple branches
